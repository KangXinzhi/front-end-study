```
0.5 - 0.4 === 1 //false
```

为什么为false呢？

因为计算机存储都是以二进制存储的。  

规则： 乘2取整顺序排列
比如0.25 在计算机中的存储

0.25 * 2 = 0.5  取整 取0
0.5  * 2 = 1.0  取整 取1

所以0.25在计算机中的存储为 01

再比如 0.2

0.2 * 2 = 0.4  取整 取0
0.4 * 2 = 0.8  取整 取0
0.8 * 2 = 1.6  取整 取1
0.6 * 2 = 1.2  取整 取1
0.2 * 2 = 0.4  取整 取0
0.4 * 2 = 0.8  取整 取0
0.8 * 2 = 1.6  取整 取1
0.6 * 2 = 1.2  取整 取1
....

所以0.25在计算机中的存储为 00110011...
丢失精度了
